# Create: Colorful Catalysts
Are you tired of your fluids spilling everywhere when trying to set up a bulk dyeing process?
This add-on solves your problems with 16 handy new blocks. They work exactly as the fluids and can be used for bulk dyeing, but you don't need to worry about spilling liquids everywhere.

[Demonstration of the catalysts. Two colored fluids on the left can be seen spilling out. The right side has two catalysts, which have the same behavior as fluids but don't spill out.](https://cdn.modrinth.com/data/cached_images/d1f99a52911061a18fb5f4114c287d5f0bd4f554.png)

## How it works
1. Obtain a bucket of dyed mastic resin (Create: Garnished) or a liquid dye bucket (Create: Dragons Plus)
2. Get any leaf block for however many catalysts you need
3. Use the bucket(s) on the leaves to turn them into catalysts
4. Place the catalysts in the airstream of a mechanical fan

## Additional Feature
One thing that always annoyed me was the inability to make a concrete powder which is dyed via bulk coloring, without the need of constant new dye.
To solve this issue, this mod has a new item called "Concrete Powder", which is used for exactly that.
This also includes conversion of colors from all the different concretes.

### Previously
As of Version 2.0.0, the mod has been simplified and doesn't rely on Create: Connected anymore.
This was done as with approval of the C:C developer, as coloring catalysts have been added to their mod natively; making this mod redundant.
However, as I've had previous requests to make this mod independent, I've used this situation as an opportunity and taken that step.

The recipes from this mod will be disabled if Create: Connected is detected.
